# Calculator Project

A responsive calculator capable of performing basic arithmetic operations 
with an interactive UI and smooth user experience.

## About

- Built a responsive calculator capable of performing basic arithmetic operations.
- Implemented interactive UI with button-based input and real-time result display.
- Focused on clean design and smooth user experience.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus

## Technologies Used

- HTML / CSS / JavaScript
